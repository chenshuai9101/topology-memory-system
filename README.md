# Topology Memory System 🕸️

> 基于拓扑结构的智能上下文管理系统 — 为 AI 助手提供无限上下文能力

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 概述

通过拓扑结构组织和管理 AI 对话上下文，突破传统 token 限制，实现更高效的信息检索和关联。

## 核心概念

- **拓扑节点** — 信息单元以节点形式存储
- **关联边** — 节点间通过语义关联连接
- **动态激活** — 根据当前对话动态激活相关节点
- **无限上下文** — 理论上支持无限量的上下文信息
